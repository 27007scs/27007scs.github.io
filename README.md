# Human Rules for Machine Power

Static advocacy website about AI laws, designed to publish easily with GitHub Pages.

## Preview locally

1. Open a terminal in `C:\Users\43768488\OneDrive\Documents\project-bolt-sb1-6scv4gom\Github`.
2. Run `npm run preview`.
3. Open `http://localhost:3000` in your browser.
4. Refresh the page after edits to see the latest changes.

## Publish on GitHub Pages

1. Push these files to a GitHub repository.
2. In GitHub, open `Settings` > `Pages`.
3. Set the source to `Deploy from a branch`.
4. Choose your main branch and `/ (root)` folder.
5. Save, then wait for GitHub Pages to build.

## Files

- `index.html` contains the page structure and copy.
- `styles.css` contains the visual design and responsive layout.
- `script.js` adds reveal animations and a simple campaign CTA message.
